Bubble Sort Visualizer

Interactive demonstration of bubble sort. Generate an array, then sort it step-by-step or animate the sort.

Comparisons: 0  |  Swaps: 0

Addendum — Project Description

Problem solved: Implemented a bubble-sort visualizer that demonstrates comparisons and swaps, allows stepping through single operations, animating the full sort, shuffling and re-generating arrays, and shows live counts of comparisons and swaps.

Approach: A modular JS file generates an array, renders bars into the DOM, and runs a generator-based bubble sort algorithm so we can pause/resume/step. The animation is driven by `setTimeout` for simple speed control.

Difficulties & solutions: 1) Keeping DOM updates smooth — solved by batching DOM updates and only re-rendering affected bars. 2) Stopping mid-animation — solved by tracking a `running` flag and clearing scheduled timeouts. 3) Accessibility — added aria-live region and labels.
