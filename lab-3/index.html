This page uses a custom font called "Creepster" that is not installed on your computer. Your browser is able to display it because I am serving it directly from Google's servers. This is one of the easiest ways to ensure your website has a unique look, regardless of what fonts a visitor has on their own system.

The Process

First, I went to a font hosting service (Google Fonts) and chose this font. The service provided an HTML <link> tag, which I added to the <head> of this document. This link tells your browser where to download the font files.

Next, in my CSS file, I used the font-family property to apply "Creepster" to the text on this page. I also included a fallback font (cursive), which your browser would use just in case it couldn't download the custom font for some reason.
